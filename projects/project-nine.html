Out of Formation

Stay with the group. A keyboard-only Micro Jam prototype about isolation, migration, formation, obstacles, and a distance-based fail meter.

Category: Game Jams and Experiments | Role: Programmer | View Repo

About the Jam

Out of Formation was created for Micro Jam 053, a 48-hour online game jam with the theme Isolation and a keyboard-only input requirement. The project was developed with one classmate, and I handled the programming role.

The player controls an animal that is constantly moving forward and must stay close to its migrating group. Obstacles can break the formation, and an isolation meter fills when the player falls too far away. If the meter reaches the limit, the player loses.

My Contribution

I implemented the main gameplay systems and connected them into a complete playable loop. I built the camera logic so it followed the migrating group and forced the player to keep up using only keyboard input.

I created the isolation system that measured distance from the group, displayed it through the isolation meter, controlled the screen dimming feedback, and triggered the lose state. I also set up the UI, menu flow, and obstacle behaviour for two different animal sections.

Technical Notes

  • Group-following camera logic tied to forward movement.
  • Distance-based isolation meter.
  • Screen dimming feedback linked to isolation level.
  • Lose state triggered when the meter reaches its limit.
  • Obstacle behaviour for two animal sections.
  • Sound manager and sound triggers supported with Bezi AI for the jam requirement.
  • Scope was kept tight by cutting the planned third animal section.
  • Remaining issue: the menu still used mouse-click navigation, which did not fully match the keyboard-only requirement.

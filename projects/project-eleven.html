Colosseum Ascendant

Fight for the mask. A Global Game Jam 2026 2.5D colosseum side-scroller with weapons, abilities, enemies, bosses, and mask fragments.

Category: Game Jams and Experiments | Role: Gameplay Programmer and Git Manager | View Repo

About the Jam

Colosseum Ascendant was created during Global Game Jam 2026 at university. The theme was Mask, and the final team had three artists and two programmers.

The game is a 2.5D side-scroller set in a colosseum. The player chooses one of three weapons, fights through waves of enemies, defeats two bosses, collects broken mask pieces, and escapes. Midway through the game, the player selects an ability, with certain ability and weapon combinations giving bonus damage for the final boss.

My Contribution

My role covered several connected gameplay systems. I worked on enemy attack behaviour, enemy types, player controls, interaction prompts, trigger volumes, item pickup, inventory state for collected mask fragments, gameplay HUD behaviour, and end screens.

I also maintained the Git repository during the jam, including branch merges and conflict resolution. Toward the end, I handled the Windows build pipeline, packaged the submission, integrated prefabs, and checked scene stability so final changes did not break the build.

Technical Notes

  • Enemy attack behaviour and multiple enemy type setup.
  • Player control implementation and interaction flow.
  • Prompt, trigger volume, and item pickup systems.
  • Mask fragment inventory state.
  • HUD, interaction prompts, and end screen separation to prevent UI overlap.
  • Git branch merging and conflict resolution.
  • Windows build packaging and submission preparation.
  • Improved mask piece feedback with sound and drop offset.
  • Tracked a remaining grounded-state animation issue after enemy interaction.

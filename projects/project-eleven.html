Colosseum Ascendant

Global Game Jam 2026 prototype. A 2.5D colosseum side-scroller built around masks, weapons, abilities, enemies, and bosses.

Category: Game Jams and Experiments | Role: Gameplay Programmer / Git Manager | GitHub

About the Jam

Colosseum Ascendant was created during Global Game Jam 2026 at university. The theme was Mask, and the team expanded to five people: three artists and two programmers.

The game is a 2.5D side-scroller set in a colosseum. The player chooses one of three weapons, fights enemy waves, defeats two bosses, collects broken mask pieces, and escapes. Midway through the game, the player chooses an ability, with certain weapon and ability combinations giving bonus damage for the final boss.

My Contribution

  • Implemented enemy attack behaviour and enemy types.
  • Worked on player controls and the interaction system.
  • Built interaction prompts, trigger volumes, and item pickup logic.
  • Managed the inventory state for collected mask fragments.
  • Kept gameplay HUD, interaction prompts, and end screens from overlapping.
  • Maintained the Git repository, branch merges, and conflict resolution.
  • Handled Windows build packaging, prefab integration, and scene stability checks before submission.

Technical Notes

  • Fixed system mismatch issues caused by older script references being left on some objects.
  • Improved mask piece feedback by adding a sound effect and drop offset so the item was easier to notice.
  • Tracked a remaining grounded-state animation issue where the player could occasionally stay stuck in a mid-air state after enemy interaction.
  • This jam showed better scope control because the team agreed on the loop early and moved systems forward instead of over-polishing one mechanic.

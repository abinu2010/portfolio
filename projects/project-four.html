SpiderSquish

Dash to defeat. A top-down 2D Unity game built using OOP principles and modular systems.

Role: Developer  •  GitHub Repository

About the Game

SpiderSquish is a top-down 2D Unity game built for my Object-Oriented Programming coursework. The goal is to move through the forest, defeat enemies by dashing into them, collect key items, and escape.

Enemies create pressure using projectiles and different behaviours. Spawners add variety by spawning enemies with different movement and attack patterns, so the player can’t solve every encounter the same way.

The game includes power-ups that increase health, plus UI that tracks score and health in real time. The design focus was to keep the core loop readable: dash, fight, collect, survive, and progress.

My Contribution

I implemented player movement, dash mechanics, and health management, then connected those systems to UI elements including a health bar and score tracker. I also built enemy spawners and enemy logic so encounters vary across the level.

I structured the codebase using OOP principles including encapsulation, abstraction, inheritance, and polymorphism to keep systems modular and maintainable. This project was used specifically to demonstrate clean system design, not just gameplay.

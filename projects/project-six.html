AI Stealth Heist Prototype

Multi-guard stealth AI. A Unity 3D prototype focused on layered perception and FSM coordination.

Role: Developer (AI / Gameplay)  •  GitHub Repository

About the Prototype

This is a Unity 3D stealth-heist prototype built for my AI in Games module, focused on believable multi-guard behaviour. Guards are driven by a finite state machine with clean transitions across patrol, suspicion, investigation, search, chase, and attack while navigating using Unity NavMesh.

Detection is layered to create stronger stealth gameplay signals: direct line-of-sight checks for visual spotting, sound events for player movement and distractions, and a decaying scent trail that leaves temporary evidence of where the player has been. Guards share alerts so when one agent escalates, nearby agents can converge on the last known location instead of acting in isolation.

The prototype includes player health with UI feedback, bait throwing to redirect guard routes, and debugging visualisation to support tuning and demonstration of perception and state changes.

My Contribution

I implemented the guard FSM architecture, NavMesh-driven movement, and the three-channel perception system (vision, sound, scent). I built the shared alert and coordination layer, last-known-position behaviours, and the search logic so guards can sweep areas rather than instantly snapping back to patrol.

I also implemented player interaction systems that drive AI responses (bait and distractions), plus UI feedback and debug tools to make the AI readable during testing and assessment.

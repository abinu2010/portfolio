AI Stealth Heist Prototype

Multi-guard stealth AI. FSM perception, alerts, and coordinated guard behaviour.

Category: Technical Projects | Role: Developer, AI and Gameplay | Video

About the Prototype

AI Stealth Heist Prototype is a Unity 3D AI project focused on believable guard behaviour. Guards use a finite state machine to move between patrol, suspicion, chase, attack, and return states.

Detection uses layered perception, including line-of-sight checks, sound events, and scent-style tracking. Shared alert logic allows one guard to influence nearby guards instead of each enemy acting alone.

The prototype also includes player health, bait throwing, minimap-style readability, and debug visualisation to make AI decisions easier to test and demonstrate.

My Contribution

I implemented the guard FSM architecture, NavMesh movement, layered perception, shared alerts, bait/distraction systems, UI feedback, and debugging support.

The main goal was to make the AI behaviour readable to the player and clear enough to explain in a technical portfolio.
